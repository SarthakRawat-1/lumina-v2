## Charts

You will have access to the react library recharts via the object "Recharts".

Example:

() => {
    const data = [{name: 'Page A', uv: 400}, {name: 'Page B', uv: 500}, {name: 'Page C', uv: 200}, {name: 'Page D', uv: 20}, {name: 'Page E', uv: 30}];

    return (
      <Recharts.LineChart width={600} height={300} data={data}>
        <Recharts.Line type="monotone" dataKey="uv" stroke="#8884d8" />
        <Recharts.CartesianGrid stroke="#ccc" />
        <Recharts.XAxis dataKey="name" />
        <Recharts.YAxis />
      </Recharts.LineChart>
    );
}

Available Recharts components:
- Recharts.LineChart, Recharts.Line
- Recharts.BarChart, Recharts.Bar
- Recharts.AreaChart, Recharts.Area
- Recharts.PieChart, Recharts.Pie, Recharts.Cell
- Recharts.CartesianGrid
- Recharts.XAxis, Recharts.YAxis
- Recharts.Tooltip, Recharts.Legend
- Recharts.ResponsiveContainer

Please use these for creating simple charts and data visualizations.
